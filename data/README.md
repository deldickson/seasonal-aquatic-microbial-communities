# Data

This project uses publicly available data from the National Ecological Observatory Network (NEON).

**Data Product**

DP1.20141.002 – Surface Water Microbe Community Taxonomy

Raw sequencing data are **not included** in this repository because they are publicly available and exceed GitHub file size limits.

The analyses in this repository were conducted using samples from:

- Posey Creek (POSE)
- Lewis Run (LEWI)

Years included in the final analyses:

- 2019
- 2021
- 2022
- 2023
- 2024

The 2018 and 2020 datasets were excluded from the final analyses because of incomplete seasonal coverage after quality control.
